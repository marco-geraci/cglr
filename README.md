# cglr
Cylindrical generalized Laplace regression: This R code implements generalized Laplace regression for cylindrical responses. It should be considered experimental. The methods are described in Geraci (2025) and Geraci and Farcomeni (2026). A real-data example is provided in the file 'turbine.R' with dataset 'turbine.csv'. The code also provides functions to fit the generalized Laplace, the projected generalized Laplace, and the projected normal densities.

Geraci M (2025). Estimation of the generalized Laplace distribution and its projection onto the circle. _Statistics and Probability Letters_, 226, 110460.
Geraci M and Farcomeni A (2026). Generalized Laplace regression for cylindrical responses with an application to road traffic accidents. _Journal of the Royal Statistical Society C_ (doi: 10.1093/jrsssc/qlag021) (Special Issue: Directional Statistics).
